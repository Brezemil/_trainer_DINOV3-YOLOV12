# Operators#

## `search_by_keyword`#

**Description** : Search for samples by keyword

**Inputs** :

  * `query`: The keyword or keyphrase to search for

  * `field`: The field to search in (any `StringField`)

  * `case_sensitive`: Whether to match case sensitively
