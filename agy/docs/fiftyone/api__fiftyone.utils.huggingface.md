    * a glob pattern of label field(s) to export

    * a dictionary mapping frame label field names to keys to use when constructing the frame label dictionaries to pass to the exporter

  * **token** (_None_) ГўВЂВ“ a Hugging Face API token to use. May also be provided via the `HF_TOKEN` environment variable

  * **preview_path** (_None_) ГўВЂВ“ a path to a preview image or video to display on the readme of the dataset repo.

  * **chunk_size** (_None_) ГўВЂВ“ the number of media files to put in each subdirectory, to avoid having too many files in a single directory. If None, no chunking is performed. If the dataset has more than 10,000 samples, it will be chunked by default to avoid exceeding the maximum number of files in a directory on Hugging Face Hub. This parameter is only applicable to :class:fiftyone.types.dataset_types.FiftyOneDataset datasets.

  * **data_card_kwargs** ГўВЂВ“ additional keyword arguments to pass to the DatasetCard constructor
